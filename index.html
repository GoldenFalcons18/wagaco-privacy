Privacy Policy / プライバシーポリシー

English

This extension ("Wagaco Auto Login") does not collect, transmit, or share any user data with external servers.

All data, including authentication information such as user ID, school ID, and password, is stored locally on the user's device using Chrome storage.

The stored data is used solely for the purpose of providing the auto-login functionality and is only accessed when the user explicitly initiates a login action.

This extension does not track browsing history, monitor user activity, or collect any personal information unrelated to its core functionality.

No data is sold, shared, or transferred to third parties.

This extension operates only on https://school.wagaco.jp/ and does not interact with other websites.

Users are responsible for managing their own credentials when using this extension.

日本語

本拡張機能（Wagaco Auto Login）は、ユーザーデータを外部サーバーへ収集・送信・共有することはありません。

ユーザーID、校舎ID、パスワードなどの認証情報は、Chromeのストレージ機能を利用してユーザーの端末内にのみ保存されます。

これらの情報は自動ログイン機能を提供する目的にのみ使用され、ユーザーが明示的にログイン操作を行った場合に限り利用されます。

本拡張機能は、閲覧履歴の追跡、ユーザーの操作監視、または機能に関係のない個人情報の収集は行いません。

ユーザーデータが第三者に販売・共有・提供されることは一切ありません。

本拡張機能は https://school.wagaco.jp/ のみで動作し、その他のサイトには影響を与えません。

本拡張機能の利用にあたっての認証情報の管理は、ユーザー自身の責任で行ってください。
